# Code of Conduct

This project follows a Code of Conduct to ensure an open and welcoming environment.

## Expected behavior
- Be respectful and considerate in all interactions.
- Use inclusive language; be mindful of diverse backgrounds.
- Report unacceptable behavior to the project maintainers.

## Unacceptable behavior
- Harassment, discrimination, or abusive language.
- Sharing private information without consent.
- Trolling or disruptive behavior.

## Enforcement
Maintainers will review reports and take appropriate action, which may include warnings, temporary bans, or removal from the project.

## Contact
If you experience or witness unacceptable behavior, please contact the maintainers at pavankumar.career@gmail.com.
